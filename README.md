MedDose - це медичний довідник, який дозволяє користувачу шукати інформацію про лікарські препарати, переглядати їхній опис та перевіряти сумісність двох ліків. Програма працює з локальною базою даних, має зручний графічний інтерфейс та забезпечує швидкий доступ до ключових даних про медикаменти без необхідності підключення до інтернету.
